## MEMBERS ONLY

Authentication project that allows users to post anonymous messages to non-members. After entering the correct secret passphrase, members are able to see the author of the post. 
Admins are able to delete posts and users, as well as update information. 

# TECH STACK
* JavaScript / Node
* PostgresSQL
* EJS (for server-side rendering)
* Passpostjs (for authentication) 
* TailwindCSS
